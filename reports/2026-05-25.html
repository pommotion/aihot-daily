Claude Code 自动模式：多任务并行的关键技巧

Boris Cherny (@bcherny)

Anthropic 工程师 Boris Cherny 分享了他使用 Claude Code 的核心技巧：开启自动模式（Auto Mode）。自动模式消除所有权限提示，是实现「多个 Claude 会话并行」的关键构件。启动一个会话后，在其运行的同时并行处理另一个会话，大幅提升单人开发吞吐量。

从「一个 AI 助手」到「一群 AI 助手」，自动模式是通往 AI-native 工作流的第一道门槛。
