pxpipe：通过图像化压缩输入 token 降低 Claude Code 成本
GitHub / Hacker News · 2026-07-03
pxpipe 将系统提示、工具文档和历史记录等密集文本渲染为 PNG，利用图像 token 按像素计价的特性压缩上下文。在 Fable 5 上约 25k 文本 token 压缩为约 2.7k 图像 token，端到端账单降低 59%-70%，SWE-bench Pro 成本约降 60%。
这是对 API 计费模型的直接 Hack，视觉 token 定价与文本不同，意味着压成本的新思路会快速普及，但图像化会让日志可读性变差，适合自动化流水线而非人工调试。

Simon Willison 博客 · 2026-07-03
与 Anthropic 交流后，建议让 Fable 用自己的判断力工作而非硬性规定行为，并让 Fable 将小任务委托给低功耗模型（Sonnet 实现、Haiku 机械修改），主循环保留判断力。

NVIDIA 联合多所大学提出 ASPIRE：自我改进机器人框架，零样本成功率最高提升 77 分
MarkTechPost · 2026-07-04
ASPIRE 通过协调器-执行器架构、闭环执行引擎和进化搜索，让机器人持续自我改进。使用 Claude Code（Claude Opus 4.6，1M token 上下文）编写控制程序，在 LIBERO-Pro 上最高比最强基线提升 77 分，双手交接成功率从 20% 提升至 92%。
这是语言模型首次被作为机器人控制程序员在框架层深度集成，1M 上下文让一次生成复杂、多步骤的控制程序成为可能，机器人领域的 AI 应用门槛正在快速降低。

我国研制全球首款基于可控存内计算的忆阻器神经动力学芯片
IT之家 · 2026-07-04
北京大学联合中科院上海微系统所发布全球首款基于可控存内计算的忆阻器神经动力学芯片，单步运算时延 2.12 毫秒，脑皮层重建任务较 GPU 提速 50-478 倍。40nm 工艺，芯片面积 0.28 mm²，运行频率 50 MHz。成果 7 月 3 日发表于《科学》。
存内计算（PIM）把存储和计算放在同一单元，从架构上解决内存墙问题；这篇发在《科学》的工作意味着中国在神经形态芯片这条替代 GPU 的路径上有了第一块标杆产品。
