通义千问发布 Qwen-VLA（Vision-Language-Action）模型，将视觉理解、语言推理与动作控制统一在一个框架内。这标志着大模型从「看图说话」迈向了「看懂世界并动手操作」的新阶段。
理解不等于行动。VLA 模型是走向具身智能的关键一步——模型不再只是回答问题，而是开始操控物理世界。

小米开源 ControlFoley，可控视频音效生成达开源 SOTA
IT之家 · 原文链接
小米大模型应用团队发布开源可控视频音效生成模型 ControlFoley，统一支持文本引导、文本控制和参考音频控制三类视频配音任务，在 VGGSound-Test 等多个基准上取得开源最优表现，代码、模型权重和 Demo 均已开放。
视频配音长期依赖人工 Foley 师。ControlFoley 把这件事变成了可Prompt控制的流程，对短视频创作者和独立制片人意义重大。

OpenRouter 支持 apply_patch，任意模型可编辑文件
OpenRouter @ X · 原文链接
OpenRouter 新增 apply_patch 服务器工具，允许任何模型通过 Responses API 使用 V4A diffs 提出文件编辑建议。模型生成补丁（创建、更新或删除文件），OpenRouter 在服务器端验证 diff 语法后执行。
这降低了 Agent 编程的门槛——不再需要模型原生支持工具调用，任何模型都能通过统一 API 修改文件。对多模型工作流是重大利好。

Adam's Law：用高频词写 Prompt，效果更好
Berry Xia @ X · 原文链接
FaceMind 团队用 100 种语言和四大核心任务实验发现，在语义不变的前提下，使用预训练语料中出现频率更高的词汇撰写提示词，可显著提升大语言模型表现。这被总结为 Adam's Law（文本频率定律）。
一个反直觉的发现：Prompt 不需要「聪明」，需要「常见」。模型在熟悉的概率空间里工作得更好——这对 Prompt 工程是个实用修正。
