Claude Blog

OpenAI Developers
OpenAI 为 Codex 推出 Python SDK（pip install openai-codex），开发者可直接将顶级编程和生图 Agent 嵌入自己的应用中，并可复用 Codex 登录态。同时发布的 Sites 功能允许 Codex 将工作、想法和计划转化为交互式网站或应用，供团队通过 URL 探索和分享。
