阿里巴巴开源 Page Agent：纯文本操控网页 DOM 的 JavaScript 库
MarkTechPost · 07/02
阿里巴巴发布 Page Agent，一个开源的 JavaScript 客户端库，通过自然语言指令直接操作 DOM 元素。与 Playwright 等外部自动化工具不同，Page Agent 将实时 DOM 脱水压缩为 FlatDomTree 文本映射，纯文本模型即可精准执行点击、表单填写等操作，无需截图或多模态模型。
不依赖视觉模型，纯文本搞定 GUI 操控——Page Agent 用 FlatDomTree 思路大幅降低了 Agent 操控网页的门槛和成本。对想做浏览器 Agent 的团队来说，这是值得关注的技术路线。
